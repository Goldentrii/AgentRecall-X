# Codex Integration

Use these natural language phrases in Codex to trigger AgentRecall MCP tools:

| Say this in Codex | AgentRecall tool |
| --- | --- |
| "Show my AgentRecall projects" | `project_board` |
| "Start project <name or number>" | `session_start` |
| "Save this session" | `session_end` |
| "Remember this decision" | `remember` |
| "What did we do last time?" | `recall` |
| "Search memory for <topic>" | `memory_query` |
